REST API Calls

lost_key
returns the lost API key
activate_user
adds user permission, or creates a new user
suspend_user
revokes user permission/access
list_products
lists all products matching vendor, description, and compartments

add_products
adds a list of new products with elements: vendor, description, alt_description, and compartments
add_asset
adds a singular new asset with elements: vendor, description, compartments, and facility
